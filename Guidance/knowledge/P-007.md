# 分类决策树 Prompt 设计

## 场景
需要 LLM 将输入归入 N 个预定义类别之一。

## 正确做法
1. 按优先级 1->N 逐条排列决策规则
2. 每条 1 行 + 关键词列表
3. 总长控制在 ~110 行以内
4. 最后一条兜底规则明确："当以上都不匹配时，选 X"
5. 每条规则用具体关键词而非抽象描述

## 核心原则
- 决策树 > 自由分类：给 LLM 明确的匹配路径，减少"偷懒选默认"的概率
- 关键词 > 描述：具体词汇比抽象定义更容易让 LLM 匹配准确
- 长度控制：Prompt 过长反而降低准确率（300 行 -> 110 行后准确率大幅提升）
