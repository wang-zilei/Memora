# CSS line-clamp 精确截断 (P-012)

- **场景**: 用 `-webkit-line-clamp` 限制多行文本时，容器 `flex: 1` 或剩余空间大于文本实际高度会导致第 N+1 行顶部像素外露
- **做法**: 对每个使用 `-webkit-line-clamp` 的元素加 `max-height: N_lines × line-height × 1em`，其中 `em` 相对元素自身 `font-size`，在不同断点下自动适配
- **原因**: `-webkit-line-clamp` 仅隐藏超出行数的文字，不限制元素高度；`flex: 1` 撑大容器后漏出下一行的顶部像素。`max-height` 精确按行高截断
- **示例**: `font-size: 13px; line-height: 1.52; -webkit-line-clamp: 3;` → `max-height: 4.56em; /* 3 × 1.52 */`
