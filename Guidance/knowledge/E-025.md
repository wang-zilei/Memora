# AI 输出残留 markdown 格式符号

## 现象
narrative 中出现字面 `\n` 字符、`**粗体**`、`## 标题` 等 markdown 残留。

## 根因
三层缺失：Prompt 未禁止 markdown -> 清洗未覆盖 markdown 残留 -> 前端未兜底渲染。

## 修复
新增 `sanitizeContent()` 7 步正则质检函数，在 capture -> Pipeline -> 前端三层链路复用：
1. 移除 markdown 标题 (#)
2. 移除加粗/斜体 (**/__)
3. 移除列表符号 (-/*)
4. 移除代码块 (```)
5. 移除字面换行符 (\n/\r)
6. 移除链接格式 [text](url)
7. 标准化空白符

## 教训
AI 输出质量需要多层兜底：Prompt 约束是第一道防线，代码质检是第二道防线，前端渲染是第三道防线。单一防线不可靠。
