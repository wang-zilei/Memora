# 语义去重 (切分+去重双引擎)

## 场景
话题切分不可靠，LLM 容易把"不同讨论角度"当作"话题转变"，需要搭配去重引擎。

## 正确做法
7 条综合判断规则（由严到松）：
1. original_question Jaccard >= 0.7 + 标题相似度 >= 0.3 -> 判重
2. 标题 Jaccard >= 0.6 + 问题相似度 >= 0.3 -> 判重
3. 标题包含 + 问题包含 -> 判重
4. narrative 前 200 字重度重叠 >= 0.65 -> 判重
5. 同 capture 会话中 card_type 相同 + 标题相似 >= 0.5 -> 判重
6. 同 capture + 问题关键词重叠 >= 80% -> 判重
7. 兜底：不判重（宁可多保留，不可漏掉）

## 核心原则
- 切分和去重是双引擎：切分负责"拆开"，去重负责"合并回去"
- 去重规则从严格到宽松，逐级降级
- 兜底偏向"不判重"，让用户手动处理
